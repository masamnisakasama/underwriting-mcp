# Troubleshooting

- `UPLOAD_TOKEN_EXPIRED`: create a new case and upload again.
- `UPLOAD_TOKEN_ALREADY_USED`: create a new case; upload tokens are single-use.
- `UNSUPPORTED_FILE_TYPE`: use an unencrypted PDF with `application/pdf`.
- `PAGE_LIMIT_EXCEEDED`: keep each file within 20 pages and the case within 30 pages.
- `MISSING_REQUIRED_DOCUMENT`: include the application form, medical disclosure, and health check PDFs.
